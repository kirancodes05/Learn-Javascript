" - Javascript is a dynamically typed language - it allows the the type of variable in the Runtime
- Variable is a container that stores a value. 
- Value of a javascript variable can be changed durig the execution of a program.

"#Rules for variable names"
1. Letters, digits, underscores and $ sign allowed
2. Must begin with with a $, _ or a letter
3. Javascript reserved words cannot be used as a variable names
4. Javascript is case sensitive

"#Let vs var and Const"
- var : globally scoped
  let and const: block scoped
- var can be updated and redeclared within its scope
- let can be updated but not redeclared
- const can neither be updated no re-declared
- New codes usually use 'let' only and not 'var' "
